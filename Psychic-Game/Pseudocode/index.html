The Psychic Game Pseudocode

1. User guesses a number that the computer will randomly choose.

2. Computer chooses a letter from a through z.

3. Based on if the user chooses the random letter selected by the computer, the choice will determine if the user wins or not.

  • The first line "Guess What I'm Tinking Of" will ask the user to guess a letter of the alphabet.

  • The "Wins" section will record the number of times that the user selects the correct letter.

  • The "Losses" section will record the number of times that the user incorrectly guesses a letter.

  • The "Guesses Left" section will notify the user how many guesses left that before they lose to the computer.

  • The "Your Guesses So Far" section will display the specific letters that the user has typed until the user either wins or loses.

a. When the user wins, the number under the wins section will increase and the game will start over again.

b. When the computer wins, the number under the losses section will increase and the game will start over again.

c. The user will be given 9 tries to correctly guess the letter.
